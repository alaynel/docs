## Welcome to fileAI!

We are an intelligent data preparation platform that transforms unstructured inputs and automates extraction into structured, machine-readable formats — all within a single workflow.

Supports single-file extraction and web data retrieval — *with cross-file extraction coming soon* — all designed for accuracy and zero hallucinations.

## About this Guide

This guide provides an overview of our API endpoints, example use cases, configuration best practices as well as frequently asked questions.

* For more code specific information on our endpoints, please refer to our[ API Reference](https://developers.file.ai/reference)
* If you need any further help, feel free to reach out to our team at [support@file.ai](https://developers.file.ai/docs/)
